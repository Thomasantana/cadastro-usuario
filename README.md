# Cadastro de Usuário - Spring Boot

Este projeto é uma aplicação **CRUD (Create, Read, Update, Delete)** desenvolvida em **Java com Spring Boot**, criada para demonstrar minhas habilidades em desenvolvimento backend e servir como parte do meu portfólio profissional.

## 🎯 Objetivo
Construir uma API REST para gerenciamento de usuários, aplicando boas práticas de arquitetura e integração com banco de dados.  
Esse projeto reflete minha dedicação em aprender tecnologias modernas e me preparar para desafios reais em empresas de tecnologia.

## 🚀 Tecnologias Utilizadas
- **Java 17**
- **Spring Boot**
- **Spring Data JPA / Hibernate**
- **Banco de Dados Relacional** (MySQL ou H2)
- **Maven**

## ⚙️ Funcionalidades
- Criar novo usuário
- Listar todos os usuários
- Buscar usuário por ID
- Atualizar dados de usuário
- Excluir usuário

## 📂 Estrutura do Projeto
- `cadastro-usuario/` → código principal da aplicação
- `.idea/` → arquivos de configuração da IDE
